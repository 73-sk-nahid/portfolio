CraftyBay - E-commerce App

  • Github URL: Project Link
  • Tech Stack: Flutter, GetX, REST APIs, Shared Preferences, MVC Architecture
  • Key Features:
    • JWT authentication via Shared Preferences
    • Product listing, cart operations, wishlist, checkout, and order placement
    • OTP verification for secure login/registration
    • Clean, minimalistic, and responsive UI design
    • Detailed product view and customer review system
  • Packages Used: flutter_lints, flutter_svg, pin_code_fields, carousel_slider, lorem_ipsum, http, logger, shared_preferences
  • Architecture: MVC with organized folder structure using GetX for state, routing, and dependency management
